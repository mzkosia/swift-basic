## 열거형
다른 언어와 다르게 강력한 기능을 가질 수 있음
대문자 카멜케이스
각 case는 소문자 카멜케이스
각 case는 그자체가 고유한 값
메서드 추가 가능

```swift
// 열거형 정의
eunm 이름 {
    case 이름1
    case 이름2
    case 이름3, 이름4, 이름5,
    ...
}
```

### 원시값
정수값 가능

정수 타입 뿐만 아니라 hoshtable 프로토콜을 따르는 모든 타입이 원시값의 타입으로 지정 가능

rawValue를 사용

case별로 각각 다른 값을 가져야 함

### 원시값을 통한 초기화
rawValue를 통해 초기화 할 수 있음

rawValue가 case에 해당하지 않을 수 있으므로 rawValue를 통해 초기화 한 인스턴스는 옵셔널 타입
